Machine learning model to recomend text

The goal in this project was develop a system to recomend text (courses on a website) using a database and machine learning techniques.

Brief description of work

The Machine Learning project for text recommendation began by studying the information and structure contained in the database related to it. Once this step was done, it was necessary to clean and adjust the format of the data to convert it into information suitable for use in the system that was being developed.

Using natural language processing (NLP) techniques, a system was built that, by means of sentences or words, performs a text search by similarity in the database and returns them in JSON format for use in Web applications.

Once the system is finished, it is deployed in the form of an API to offer the service. As a future step, in agreement with the client, it can be integrated into a web service and data retribution can be automated, without the need for user intervention.
